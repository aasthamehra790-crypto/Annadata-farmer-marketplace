# Annadata – Farmer Direct Marketplace

Annadata is a hackathon project built during the Hack the Horizon Hackathon.

## Problem
Farmers often earn less because intermediaries control pricing and distribution.

## Solution
Annadata is a mobile application that enables farmers to sell produce directly to consumers without middlemen.

## Features
- Direct farmer-to-customer marketplace
- Simple UI designed for accessibility
- Order confirmation automation

## Technologies Used
- Adalo
- Firestore
- Zapier

## Achievement
Secured 2nd Place at Hack the Horizon Hackathon.
